# RZL Hilfe

Die erste live Preview kann [hier](https://purple-wave-084806310.4.azurestaticapps.net/) gefunden werden.

## Contribute

Eine live Preview kann lokal mit diesem Befehl gestartet werdern:
```
mkdocs serve
```
