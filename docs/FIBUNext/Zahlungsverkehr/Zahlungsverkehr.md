# Zahlungsverkehr

Durch die Umstellung auf das SEPA-Format wurde ein europaweit einheitliches Format für den Zahlungsverkehr eingeführt. Transaktionen in Euro werden innrerhalb des SEPA-Raumes über ein einziges Konto, unabhängig davon ob es sich um eine nationale oder grenzüberschreitende Zahlung handelt, durchgeführt. 

!!! warning "Hinweis"

    Das Modul **Zahlungsverkehr** steht in der Firmenversion nur dann zur Verfügung, wenn es bestellt wurde.
