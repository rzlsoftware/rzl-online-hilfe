
# Fehler beim Senden einer E-Mail

## Problembeschreibung
Fehler beim Senden einer E-Mail.  
**Fehlercode: -26**

## Ursache
Dieses Problem tritt auf, wenn eine der folgenden Bedingungen vorliegt:

- In MS Windows ist kein Standard-E-Mail-Programm hinterlegt.
- Eine beschädigte Microsoft Outlook-Installation liegt vor.
- Die Windows-Funktion "Senden an - E-Mail-Empfänger" funktioniert nicht.

## Lösung
Wenden Sie folgende Methode(n) an, um das Problem zu beheben:

!!! warning "Hinweis"
    Seit 2023 installiert Microsoft mit automatischen Windows-Updates das neue Outlook („New Outlook") auf vielen Geräten und setzt es oft als Standard-E-Mail-App – auch wenn das klassische Outlook oder andere Programme bevorzugt werden. Seit Februar 2025 wird es mit monatlichen Updates flächendeckend ausgerollt und ersetzt die alten Mail- & Kalender-Apps. Über Einstellungen im klassischen Outlook oder Gruppenrichtlinien kann das jedoch deaktiviert werden.

- Hinterlegen Sie Microsoft Outlook als Standard-E-Mail-Programm.
- Führen Sie eine Reparatur der Office-Installation durch.
